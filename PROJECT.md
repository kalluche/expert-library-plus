# 专家库 + 人名增强方案 - GitHub 开源项目

## 📋 项目背景
基于 Claude Skills 的启发，创建一个开源的专家库系统，通过"人名作为质量锚点"的概念提升AI任务执行质量。

## 🎯 项目目标
- 将现有的 OpenClaw 专家库系统开源
- 添加人名增强功能的目录结构和示范
- 提供用户自建人名库的完整指南
- 建立完整的 GitHub 开源项目标准结构
- 支持中英文双语，面向全球开发者

- **项目维护者**: 用户
- **技术顾问**: Qwen-Max

## 📁 最终交付物位置
所有要上传到 GitHub 的内容都将放在本项目目录下的 `github-upload/` 文件夹中。

## 🔄 双语同步机制
- **文件结构**: 主要文档分离（README-en.md/README-zh.md），指南类文档合并（双语格式）
- **更新规则**: 任何内容更新必须同时修改中英文版本
- **检查工具**: 提供 `bilingual-sync-check.sh` 脚本验证同步状态
- **维护责任**: 项目维护者确保双语内容一致性
