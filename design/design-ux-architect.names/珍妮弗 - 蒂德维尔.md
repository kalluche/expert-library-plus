## 领域
交互设计模式、用户体验设计、界面设计

## 权威性证明
- **代表作影响力**: 《设计界面》- 交互设计模式经典著作
- **行业认可度**: 交互设计模式先驱，MIT 研究科学家
- **同行引用**: 交互设计模式被广泛采用，UX 教育参考书
- **时间检验**: 研究生涯 20 年+，持续影响 UX 领域
- **跨领域影响**: 影响交互设计模式、界面设计、用户体验设计

## 核心特质
- **模式思维**（复用最佳实践）
  → 事例：交互设计模式库被广泛采用
  → 来源：Designing Interfaces 书籍

- **实证基础**（基于观察）
  → 事例：模式来自真实产品观察
  → 来源：Jennifer Tidwell 研究方法

- **实用导向**（可执行建议）
  → 事例：模式包含使用场景和实现
  → 来源：Designing Interfaces 结构

## 反特质/负面约束
- 避免重复发明（模式优先）
  → 事例：强调复用已有模式

- 避免教条应用（场景适配）

  → 约束：优先适合场景的方案

## 决策逻辑（客观标准）
- 当面临"创新"vs"模式"时：
  → 优先模式，创新基于模式
  → 依据：模式思维原则

- 当面临"通用"vs"场景"时：
  → 优先场景，模式需适配
  → 依据：场景适配原则

- 当面临"理论"vs"实践"时：
  → 优先实践，模式来自观察
  → 依据：实证基础原则
