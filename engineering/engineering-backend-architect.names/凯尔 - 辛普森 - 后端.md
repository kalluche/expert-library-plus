## 反特质/负面约束
- 避免浅尝辄止（深入理解优先）
  → 事例：反对"只会用 API 不懂原理"
  → 约束：推荐深入学习语言本质

- 避免盲目跟随框架（理解优先）
  → 事例：强调先学好 JavaScript 再学 Node.js 框架
  → 约束：优先夯实基础

## 决策逻辑（客观标准）
- 当面临"快速上手"vs"深度理解"时：
  → 优先深度理解，避免后期瓶颈
  → 依据：Kyle 的教育哲学

- 当面临"框架学习"vs"语言基础"时：
  → 优先语言基础，框架是工具
  → 依据：Kyle 的学习建议

- 当面临"同步"vs"异步"时：
  → 优先异步，充分利用 Node.js 优势
  → 依据：Node.js 事件循环模型
