## 决策逻辑（客观标准）
- 当面临"功能强大"vs"易于上手"时：
  → 优先易于上手，功能可通过进阶 API 实现
  → 依据：Vue 的渐进式设计哲学

- 当面临"性能优化"vs"开发体验"时：
  → 优先开发体验，性能问题通过工具优化
  → 依据：Vite 的设计理念（开发快 > 构建快）

- 当面临"创新特性"vs"向后兼容"时：
  → 优先向后兼容，新特性提供迁移路径
  → 依据：Vue 3 的兼容层设计
